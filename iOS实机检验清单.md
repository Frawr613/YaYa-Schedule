# 鸦鸦日程 iOS 实机检验清单

这份清单用于在 Xcode 工程完成后做真机验证。Windows 侧已经可以检查源码、资源、缓存版本、工程引用和压缩包结构，但以下项目必须在 macOS + Xcode + iPhone 真机上确认。

## 1. Xcode 工程与签名

- [ ] 使用 Xcode 打开 `YayaScheduleClean.xcodeproj`，工程能正常索引，无红色缺失文件。
- [ ] `YayaScheduleClean` 与 `YayaScheduleWidget` 两个 Target 都能选择同一个有效开发团队。
- [ ] 主 App Bundle ID 为 `com.xuyunfan.yayaschedule.clean`，Widget Bundle ID 为 `com.xuyunfan.yayaschedule.clean.widget`。
- [ ] App Group `group.com.xuyunfan.yayaschedule` 在主 App 与 Widget 两端均启用。
- [ ] 版本号显示为 `2.0`。
- [ ] 真机 Build & Run 成功，安装后桌面名称为“鸦鸦日程”。

## 2. 首次启动与基础界面

- [ ] 首次启动进入主页面，标题、今日/设置入口、玻璃卡片风格显示正常。
- [ ] 首次提示窗口会弹出，并包含“可以在设置中查看提示”的说明。
- [ ] 设置页能正常打开和关闭，浮窗出现时下层不可误触。
- [ ] iOS 端图标说明显示为“在 iOS App 内保存后会切换应用图标”，不出现 Android 误标。
- [ ] 页面文字不可被长按选中，输入框和可编辑区域除外。

## 3. 主题、模板与 UI 桥接

- [ ] 四个预设主题切换后，字体颜色、模块颜色和玻璃质感同步变化。
- [ ] 自定义主题页中预览框位于色号输入上方，保存后等待数秒能稳定应用。
- [ ] 设置、主题、自定义主题、图标、日期、时间、周次、学期选择均使用内置独立 UI 控件。
- [ ] 周次/日期切换、课程概览分页、日程概览分页的 chip 选中态有明显区分。
- [ ] 课程概览 chip 会自动让当前学期进入可见区域，但不会强制整页跳动。
- [ ] 浮窗切换和分页切换有平滑翻页效果，无左右溢出滑动网页的问题。

## 4. 本地数据与缓存

- [ ] 新建、修改、删除课程/日程/DDL 后，当前卡片、概览、搜索结果均同步刷新。
- [ ] 修改日程或特殊变更后，日程卡显示立刻反映最新状态。
- [ ] 关闭并重新打开 App 后，本地数据、主题、模板、提示状态能恢复。
- [ ] 多缓存少实时渲染逻辑有效，连续切换页面和浮窗无明显卡顿或白屏。

## 5. 常驻日程、周次与日期联动

- [ ] 添加常驻日程时，周次控件使用内置独立 UI。
- [ ] 修改开始周/结束周会联动开始日期/结束日期。
- [ ] 修改中间周次不会错误改动起止日期。
- [ ] 周次输入栏、周次 chip、起止日期、星期之间联动正确。
- [ ] 特殊变更中的移动、取消分页逻辑可用，并允许取消已设置的特殊变更。

## 6. DDL 与提醒

- [ ] 新建 DDL、追加 DDL、修改 DDL、完成 DDL、恢复/删除已完成 DDL 均正常。
- [ ] DDL 已完成分页中，内容模糊检索和时间范围检索可选可不选，组合筛选正确。
- [ ] DDL 进行中/已完成分页逻辑与原 App 体验一致。
- [ ] 新建 DDL 的提醒选择 UI 紧凑，选项状态文字颜色跟随选中态变化。
- [ ] 日程同步到 DDL 只作为显示入口，不破坏原日程提醒。
- [ ] 需要开启提醒才允许同步显示到 DDL。

## 7. iOS 通知与后台触发

- [ ] 设置中请求通知权限时，走 iOS 系统弹窗或系统设置入口。
- [ ] 通知权限允许后，权限面板显示已开启或可用状态。
- [ ] 日程提醒、DDL 提醒、追加日程提醒、追加 DDL 提醒都能各自创建系统通知。
- [ ] 第一条之后继续创建的新提醒也会挂载通知。
- [ ] 锁屏或息屏后，到点能收到通知。
- [ ] 点击通知进入 App 后，不会连续重复弹出或重复响铃。
- [ ] 有权限时通知声音存在；系统静音或通知声音关闭时，界面提示合理。

## 8. iOS Widget

- [ ] Widget 只提供较小的两个尺寸，不出现最大号组件。
- [ ] Widget 配色随当前主题变化，预设主题和自定义主题都能同步。
- [ ] 最近 DDL 显示在最近日程上方，DDL 区域较窄，最近日程区域较宽。
- [ ] Widget 不显示无效信息；无 DDL 或无日程时显示干净的空状态。
- [ ] 最近日程的竖向透明虚线能按时间进度从左向右推进。
- [ ] 虚线左侧背景会轻微变色，且不会遮挡文字。
- [ ] App 内修改日程、特殊变更、DDL 后，Widget 数据能跟随刷新。

## 9. 教务导入

- [ ] 点击教务导入后只打开一层网页，不出现双重网页。
- [ ] 信息门户打开后可以点击电脑端入口并进入信息门户。
- [ ] 自动输入账号密码功能正常，不破坏用户手动输入。
- [ ] 登录后从办事大厅进入教学管理/教务管理系统流程可操作。
- [ ] 导入课表、导入考试、返回鸦鸦三个按钮在视窗内显示完整，尺寸适合触控。
- [ ] 教务导入浮窗可拖动，可按角部自由缩放，按钮随缩放保持可用。
- [ ] 学年/学期选择和开学日期选择使用主题适配的内置独立 UI 控件。
- [ ] 返回主页面前允许多次导入，每次按选择的学期分开存储。
- [ ] 悬浮面板内完成学期选择、开学日期选择、抓取和存储，不依赖返回主页面触发回传。

## 通过标准

- 主 App、Widget、通知、教务导入四块都能在真机完成主要流程。
- 不出现 Android/iOS 平台误标。
- 不出现浮窗遮罩失效、下层误触、页面左右滑动、明显白屏或重复通知。
- 修改数据后，主页面卡片、概览、提醒和 Widget 均保持同步。
